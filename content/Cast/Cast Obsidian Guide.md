# How to use this Obsidian

This obsidian is synced between us using the obsidian git plugin. 

I have created some shortcuts for us to use. 

| Command                               | Windows          | Mac                |
| ------------------------------------- | ---------------- | ------------------ |
| Pull from remote                      | `ctrl + alt + q` | `cmd + option + q` |
| Sync (pull, commit all and then push) | `ctrl + alt + w` | `cmd + option + w` |
| Commit                                | `ctrl + alt + e` | `cmd + option + e` |
| Open git raw command                  | `ctrl + alt + r` | `cmd + option + r` |

Always try to sync as often as possible. 

When your into a conflict or error. Try do open git raw command and type `reset --hard`, this will reset all your changes and get a fresh copy from the remote repo.
